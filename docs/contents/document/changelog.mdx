# Change Log

This document provides a chronological record of all notable changes to ClickUp MCP Server.

## [0.1.0] - 2025-07-15

### Added
- Initial release of ClickUp MCP Server
- Support for HTTP streaming transport
- Support for Server-Sent Events (SSE) transport
- Basic API endpoints for Teams, Spaces, Folders, Lists, and Tasks
- Environment variable loading from .env files with custom path support
- Comprehensive test suite
- Basic documentation

### Security
- Secure token management for API authentication

## [0.0.2] - 2025-06-30

### Added
- Pre-release version with core functionality
- Initial implementation of MCP protocol
- Basic transport interfaces

### Fixed
- Resolved several critical bugs in API client implementation
- Fixed issue with connection handling in HTTP streaming

## [0.0.1] - 2025-06-15

### Added
- Initial project structure
- Basic ClickUp API client implementation
- Core models for ClickUp resources
- Project configuration and setup
